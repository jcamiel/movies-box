## Building from sources

1. Installation
    ```shell
    $ npm install
    ```
2. Build
    ```shell
    $ npm run build
    ```
3. Run
   ```shell
   $ DEBUG=movies-box:* node dist/bin/www
   ```

Then, open <http://localhost:3000>
